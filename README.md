# Subrajit Jena — Portfolio Website

> A dynamic, modern Full Stack Developer portfolio built with React + Vite.

[![Live Demo](https://img.shields.io/badge/Live-Demo-64ffda?style=flat-square)](https://subrajitjena.netlify.app)
[![GitHub](https://img.shields.io/badge/GitHub-subrajitjena-181717?style=flat-square&logo=github)](https://github.com/subrajitjena)

---

## ✨ Features

- 🎨 **Premium Dark UI** — Deep navy + teal accent color palette
- 🌌 **Particle Canvas** — Interactive constellation background in hero section
- 📱 **Fully Responsive** — Mobile, tablet, and desktop layouts
- 🔢 **Animated Skill Bars** — Progress bars that animate on scroll
- 🃏 **Project Cards** — Hover effects with top-border animations
- 📬 **Contact Form** — Client-side validation with success state
- 🔗 **Smooth Scrolling** — Seamless navigation between sections
- 🎯 **Active Nav Links** — Real-time scroll spy for the navbar
- 🔍 **SEO Ready** — Meta tags, title, and description

---

## 🛠 Technologies Used

| Layer        | Technologies                          |
|---|---|
| **Framework**  | React 18 + Vite 5                   |
| **Styling**    | Vanilla CSS + CSS Variables         |
| **Animations** | Framer Motion + CSS Keyframes       |
| **Navigation** | react-scroll (smooth Spy)           |
| **Icons**      | lucide-react                        |
| **Font**       | Poppins (Google Fonts)              |

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

---

## 🌐 Deployment

Deploy to **Netlify**:
1. `npm run build`
2. Upload the `dist/` folder to Netlify, or connect your GitHub repo

Deploy to **Vercel**:
1. Connect your GitHub repo to Vercel
2. Set framework preset to **Vite**
3. Deploy automatically on push
